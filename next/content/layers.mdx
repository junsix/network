## OSI 7계층 vs TCP/IP 4계층

OSI는 교과서적인 분류, TCP/IP는 실제로 인터넷이 동작하는 방식에 가까운 분류입니다.

<DataTable>
  <thead>
    <tr>
      <th>OSI</th>
      <th>TCP/IP</th>
      <th>실무 비고</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>L7 Application<br/>L6 Presentation<br/>L5 Session</td>
      <td>Application</td>
      <td>대부분의 앱 프로토콜이 5~7층을 한꺼번에 책임짐 (예: HTTP는 인코딩/세션/의미를 모두 다룬다).</td>
    </tr>
    <tr>
      <td>L4 Transport</td>
      <td>Transport</td>
      <td>TCP/UDP/QUIC. "호스트 안의 프로세스"를 가리키는 포트 개념이 여기서 등장.</td>
    </tr>
    <tr>
      <td>L3 Network</td>
      <td>Internet</td>
      <td>IP 주소가 핵심. "어떤 네트워크의 어떤 호스트인가."</td>
    </tr>
    <tr>
      <td>L2 Data Link<br/>L1 Physical</td>
      <td>Network Access (Link)</td>
      <td>이더넷·Wi-Fi 등 매체별 규격. 실제 비트가 흐른다.</td>
    </tr>
  </tbody>
</DataTable>

## 계층별 보안 고려사항

<DataTable>
  <thead>
    <tr><th>계층</th><th>대표 위협</th><th>대응 위치</th></tr>
  </thead>
  <tbody>
    <tr><td>L2</td><td>ARP 스푸핑, MAC flooding, 비인가 AP</td><td>스위치 보안(DAI, port security), 802.1X</td></tr>
    <tr><td>L3</td><td>IP 스푸핑, 라우팅 하이재킹, ICMP 남용</td><td>uRPF, BGP RPKI, ACL</td></tr>
    <tr><td>L4</td><td>SYN flood, 포트 스캐닝</td><td>SYN cookie, 방화벽, rate limit</td></tr>
    <tr><td>L7</td><td>SQLi, XSS, 인증 우회</td><td>WAF, 입력 검증, 인증/인가 설계</td></tr>
    <tr><td>L6/7</td><td>도청, 위변조</td><td>TLS (기밀성·무결성·인증)</td></tr>
  </tbody>
</DataTable>

<Callout>
  한 계층의 방어만으론 부족합니다. 예를 들어 TLS(L6/7)는 도청은 막지만,
  ARP 스푸핑(L2)으로 트래픽이 다른 곳으로 흐르는 것 자체는 막지 못합니다.
  그래서 <strong>심층 방어(defense in depth)</strong>가 필요합니다.
</Callout>

<Callout tone="bad">
  <strong>"몇 계층?"에 단정하지 말 것.</strong> OSI 7층은 깔끔한 칸이 아니라 *연속체*입니다.
  <ul>
    <li><strong>TLS</strong>: 종종 L6로 분류되지만 실제로는 L5(세션)~L7(애플리케이션) 경계에 걸친 별도 프로토콜. TCP 위 record layer로 동작.</li>
    <li><strong>ARP</strong>: L2 프로토콜로 분류되지만 *L3 주소를 L2 주소로 매핑*하는 가교라 <strong>L2.5</strong>로도 부름.</li>
    <li><strong>ICMP</strong>: IP 위 캡슐화 + L3 컨트롤 메시지 역할이라 <strong>L3.5</strong>로 보는 게 정확.</li>
    <li><strong>QUIC</strong>: UDP(L4) 위 신뢰성·암호화·멀티플렉싱이라 L4~L7 통합으로 봄. UDP 차단 환경에선 TCP fallback 필수.</li>
  </ul>
  면접에서 "TLS는 몇 계층?"에 단답하면 따라잡힘. *경계 모호성*을 명시하는 게 시니어 답변.
</Callout>
